KEY DESIGN DECISIONS

  • Sheet-based auth — no page navigation, everything in a sliding panel from right
  • Inline error retry — wrong code? Retry in place. Session expired? Re-auth preserves connection config
  • Auto health check — runs immediately after auth, catches issues before the user walks away
  • Visual scope picker — fetches actual group names/member counts from the connected platform (needs new API endpoint)
  • Status-first hub — connection health visible at a glance, error states surface re-auth actions
