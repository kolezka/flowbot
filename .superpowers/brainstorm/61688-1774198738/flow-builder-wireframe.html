KEY DESIGN DECISIONS

  • ⌘K command palette — fuzzy search across nodes, actions, and settings. Power users never touch the sidebar
  • Variable autocomplete — type {{"{{"}} in any field to get a dropdown with available variables + types, scoped to the current node's upstream data
  • Docked debugger — bottom panel (not overlay), resizable, with real-time WebSocket updates. Steps + variable inspector side by side
  • Auto-save drafts — saves every 10s, "Publish" creates an explicit version. Unsaved indicator always visible in toolbar
  • Node templates — pre-built flow snippets (Welcome Flow, Moderation Flow) that drop in as a group of connected nodes
  • Inline validation — each node shows valid/invalid state, property panel shows per-field validation with fix suggestions
  • Component decomposition — the 1400-line monolith splits into: NodePalette, CommandPalette, PropertyPanel, ExecutionDebugger, VersionHistory, CanvasToolbar (each <400 lines)
