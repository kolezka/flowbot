Final Architecture — Flow Context Menu

Zustand store + factory pattern with @radix-ui/react-context-menu

System Overview
┌─────────────────────────────────────────────────────────────────┐
│  createFlowContextMenu()                                       │
│  Returns: [useFlowStore, FlowContextMenu]                       │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │  Zustand Store (single source of truth)                  │    │
│  │                                                         │    │
│  │  ── Flow State ──                                       │    │
│  │  nodes: Node[]                                          │    │
│  │  edges: Edge[]                                          │    │
│  │  selectedNode: Node | null                              │    │
│  │  copiedNode: Node | null                                │    │
│  │                                                         │    │
│  │  ── Menu State ──                                       │    │
│  │  menu: { isOpen, position, context } | null             │    │
│  │                                                         │    │
│  │  ── Flow Actions ──                                     │    │
│  │  applyNodeChanges(changes)                              │    │
│  │  applyEdgeChanges(changes)                              │    │
│  │  addEdge(connection)                                    │    │
│  │  addNode(type, position)                                │    │
│  │                                                         │    │
│  │  ── Context Menu Actions ──                             │    │
│  │  deleteNode(id)    duplicateNode(id)                    │    │
│  │  copyNode(id)      pasteNode(position)                  │    │
│  │  toggleNode(id)    addStickyNote(position)              │    │
│  │  deleteEdge(id)    insertNodeOnEdge(edgeId, type)       │    │
│  │  selectAll()       fitView(fitViewFn)                   │    │
│  │                                                         │    │
│  │  ── Menu Openers ──                                     │    │
│  │  openNodeMenu(node, pos)                                │    │
│  │  openEdgeMenu(edge, pos)                                │    │
│  │  openCanvasMenu(pos)                                    │    │
│  │  closeMenu()                                            │    │
│  └─────────────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────────────┘

         │ useFlowStore()                    │ <FlowContextMenu />
         ▼                                   ▼

┌─────────────────────────┐    ┌──────────────────────────────────┐
│  FlowEditorInner          │    │  FlowContextMenu (component)       │
│                         │    │                                  │
│  • reads nodes/edges    │    │  • reads menu state from store   │
│  • passes to ReactFlow  │    │  • renders ContextMenu.Root      │
│  • calls store actions  │    │  • switches on context.type:     │
│  • NO local useState    │    │    ├─ 'node' → node menu items   │
│    for flow data        │    │    ├─ 'edge' → edge menu items   │
│                         │    │    └─ 'canvas' → canvas menu     │
│                         │    │  • calls store actions on click  │
│  ┌───────────────────┐  │    │  • zero props needed             │
│  │ FlowCanvas        │  │    │                                  │
│  │                   │  │    │  Nested submenu for "Add node":  │
│  │ ReactFlow         │  │    │  ├─ 🟢 Triggers ▸               │
│  │  onNodeCtxMenu ───┼──┼───▶│  ├─ 🟡 Conditions ▸             │
│  │  onEdgeCtxMenu ───┼──┼───▶│  ├─ 🔵 Actions ▸                │
│  │  onPaneCtxMenu ───┼──┼───▶│  ├─ 🟣 Advanced ▸               │
│  │                   │  │    │  └─ 📎 Annotation ▸              │
│  └───────────────────┘  │    │                                  │
└─────────────────────────┘    └──────────────────────────────────┘
