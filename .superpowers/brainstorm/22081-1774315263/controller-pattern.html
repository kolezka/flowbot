Context Menu Controller Architecture

Zustand controller + factory pattern — how the pieces connect

createFlowContextMenu() — Factory Output
// Factory returns a tuple: [controller, Component]

const [contextMenu, FlowContextMenu] = createFlowContextMenu()

// ─── Controller (imperative API) ───────────────
contextMenu.openForNode(node, position)
contextMenu.openForEdge(edge, position)
contextMenu.openForCanvas(position)
contextMenu.close()
contextMenu.setCopiedNode(node)
contextMenu.getCopiedNode()
contextMenu.registerActions({ onDelete, onDuplicate, ... })

// ─── Component (declarative, self-wired) ──────
<FlowContextMenu /> ← reads from same Zustand store
renders ContextMenu.Root + Content
picks node/edge/canvas variant automatically
calls registered actions on click
Usage in FlowEditorInner (edit page)
// Create once at module level
const [contextMenu, FlowContextMenu] = createFlowContextMenu()

function FlowEditorInner() {
// Register actions once (they close over component state)
useEffect(() => {
contextMenu.registerActions({
onDeleteNode: (id) => setNodes(ns => ns.filter(…)),
onDuplicateNode: (id) => { … },
onToggleNode: (id) => { … },
onDeleteEdge: (id) => setEdges(…),
onPasteNode: (pos) => { … },
onAddStickyNote: (pos) => { … },
onAddNode: (type, pos) => handleDrop(type, pos),
onSelectAll, onFitView, onInsertNodeOnEdge,
})
}, [nodes, edges, …])

// Pass controller to FlowCanvas for event wiring
<FlowCanvas contextMenu={contextMenu} … />

// Drop the component anywhere — it reads its own store
<FlowContextMenu />
}
Inside FlowCanvas — event wiring
<ReactFlow
onNodeContextMenu={(e, node) => {
e.preventDefault()
contextMenu.openForNode(node, { x: e.clientX, y: e.clientY })
}}
onEdgeContextMenu={(e, edge) => {
e.preventDefault()
contextMenu.openForEdge(edge, { x: e.clientX, y: e.clientY })
}}
onPaneContextMenu={(e) => {
e.preventDefault()
contextMenu.openForCanvas({ x: e.clientX, y: e.clientY })
}}
/>
Zustand Store Shape
{
// Menu state
isOpen: boolean
position: { x: number, y: number } | null
context:
| { type: 'node', node: Node }
| { type: 'edge', edge: Edge }
| { type: 'canvas' }
| null

// Clipboard
copiedNode: Node | null

// Registered action handlers
actions: FlowContextMenuActions | null
}
